You build workflows, connect MCP tools, create reusable skills, deploy an automation, run screenshot feedback loops, schedule routines, and finish with a personal executive assistant workspace.

## For teams

Use the course as a 7-day internal challenge:

- Day 1: Everyone builds the same workflow pattern.
- Day 2: Everyone connects a tool through MCP.
- Day 3: Each person creates one reusable skill.
- Day 4: One workflow becomes a deployed automation.
- Day 5: Teams practice screenshot-based visual iteration.
- Day 6: Teams design scheduled and monitored routines.
- Day 7: Each person creates a personal EA folder.
