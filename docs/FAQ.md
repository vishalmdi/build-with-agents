# FAQ

## Is the course free?

Yes. The 7-day course, prompts, capstone, and downloadable files are free on the official site.

Start here: [buildwithagents.vercel.app](https://buildwithagents.vercel.app/)

## Is this the full course source?

No. This is the public GitHub landing repo and walkthrough. The course experience lives on the official site.

This repo intentionally does not include:

- Private website source code.
- Full lesson source files.
- Downloadable resource bundles.
- Internal deployment configuration.
- An open-source license.

## Why keep this repo public?

GitHub is where builders discover useful projects. This repo gives Build With Agents a public face, a star target, and a shareable technical summary.

## Why is there no open-source license?

Because this is not an open-source course repo. All rights are reserved.

## Do I need to know how to code?

You need to be comfortable using VS Code and following instructions. The course is designed so you do not write code blind.

## What tools are used?

Claude Code, VS Code, Firecrawl MCP, Trigger.dev, and practical workflow files. The exact tools vary by day.

## What should I build first?

Start with Day 1. Do not skip the first workflow. The later days assume you understand the WAT model: Workflows, Agent, Tools.

## Can a team use this internally?

Yes, teams can use the official site for learning and can share links to the course. Copying, republishing, reselling, or packaging the course materials as your own is not allowed without written permission.
